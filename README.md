# Algorand Account Balance Checker

This project demonstrates a simple Algorand application using the Python SDK.

It connects to the Algorand TestNet and checks the balance of a wallet address.

## Features
- Connects to Algorand TestNet
- Retrieves account balance
- Uses Algorand Python SDK

## Requirements
- Python 3
- Algorand Python SDK

## Install Dependencies
pip install -r requirements.txt

## Run Project
python main.py
